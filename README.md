# Contributors
To edit a page:
 - Fork the repo
 - Edit the .md file (either directly on Github or locally)
 - Send a pull request.

To create a new page:
- Fork the repo
- Clone your fork
- Create a .md file in the src/swift directory (please follow the numbering convention)
- Send a pull request.
